Fonctions

Les bases sont pretes.

Inscription

Creation de compte avec validation email, pseudo et mot de passe fort.

Connexion

Verification du compte et ouverture d'une session dans ce navigateur.

Securite

Les mots de passe sont haches avec PBKDF2 via Web Crypto, jamais stockes en clair.

Important

GitHub Pages reste un site statique.

01 Ce qui marche Un visiteur peut creer un compte et se reconnecter sur le meme navigateur.
02 Ce qui est protege Le mot de passe n'est pas sauvegarde en clair dans le stockage local.
03 Pour une vraie prod Il faudra brancher Supabase, Firebase, Auth0 ou un backend avec base de donnees.
